NumPy 數值運算
建立陣列
np.array([1, 2, 3]) np.zeros((3, 4)) # 全0 np.ones((3, 4)) # 全1 np.random.rand(3, 4) # 隨機 0~1 np.random.normal(0, 1, (3,4)) # 常態分布 np.random.seed(42) # 固定隨機種子
形狀與統計
arr.shape # (列數, 欄數) arr.reshape(2, 6) arr.T # 轉置 arr.mean() # 平均 arr.std() # 標準差 arr.max() # 最大值 arr.sum(axis=0) # 沿列方向加總 arr.sum(axis=1) # 沿欄方向加總
axis 記憶法：axis=0 往「下」計算（跨列）；axis=1 往「右」計算（跨欄）。mean(axis=0) = 每欄的平均。
切片語法（常考）
arr[0, :] # 第0列，全部欄 arr[:, 0] # 全部列，第0欄 arr[:, 0:9] # 全部列，第0~8欄（不含9） arr[:, 9] # 全部列，第9欄（通常是 y 標籤） # 複合賦值運算子 X -= X.mean(axis=0) # 等同 X = X - X.mean(axis=0) X /= X.std(axis=0) # 等同 X = X / X.std(axis=0)

Sklearn 資料前處理 Preprocessing
train_test_split — 切分資料
from sklearn.model_selection import train_test_split X_train, X_test, y_train, y_test = train_test_split( X, y, test_size=0.2, # 20% 當測試集 random_state=42, # 固定隨機種子 stratify=y # 依標籤比例分層抽樣 )
StandardScaler（Z-score）
from sklearn.preprocessing import StandardScaler scaler = StandardScaler() X_train = scaler.fit_transform(X_train) X_test = scaler.transform(X_test) # 結果：平均=0，標準差=1 # 適用：有離群值、SVM、迴歸
MinMaxScaler（正規化）
from sklearn.preprocessing import MinMaxScaler scaler = MinMaxScaler() X_train = scaler.fit_transform(X_train) X_test = scaler.transform(X_test) # 結果：壓縮到 0~1 之間 # 適用：無離群值、神經網路
最常考的陷阱：StandardScaler 只能在 X_train 上 fit，X_test 只能 transform。若對 X_test 也 fit，就發生「資料洩漏」，test 的資訊混入 train。

Ridge vs Lasso：Ridge（L2）係數縮小但不為0，防過擬合；Lasso（L1）係數可歸零，自動特徵選擇。alpha 越大，懲罰越重。
評估指標 Metrics
混淆矩陣 — 四格定義
# 預測：正 預測：負 # 實際：正 TP（對的正） FN（漏掉了） # 實際：負 FP（誤報了） TN（對的負） Accuracy = (TP + TN) / 全部 Precision = TP / (TP + FP) # 預測為正，真的正的比例 Recall = TP / (TP + FN) # 實際為正，有抓到的比例 F1 = 2 × P × R / (P + R) # 調和平均
程式碼
from sklearn.metrics import ( accuracy_score, precision_score, recall_score, f1_score, confusion_matrix, roc_auc_score, mean_squared_error, r2_score ) accuracy_score(y_test, y_pred) f1_score(y_test, y_pred, average='macro') # 多分類要加 average confusion_matrix(y_test, y_pred) mean_squared_error(y_test, y_pred) mean_squared_error(y_test, y_pred) ** 0.5 # RMSE r2_score(y_test, y_pred)

垃圾郵件Precision不能把正常信誤判

不平衡資料F1 / AUCAccuracy 會失真
房價預測MSE / RMSE / R²迴歸任務
交叉驗證 Cross Validation
cross_val_score — 最常考
from sklearn.model_selection import cross_val_score, StratifiedKFold # 方法一：直接給整數（推薦，自動使用 StratifiedKFold） scores = cross_val_score(model, X, y, cv=5, scoring="accuracy") print(scores) # [0.82, 0.85, 0.80, 0.83, 0.84] print(scores.mean()) # 0.828 ← 真實模型能力 # 方法二：明確建立 StratifiedKFold（效果相同） cv = StratifiedKFold(n_splits=5, shuffle=True, random_state=42) scores = cross_val_score(model, X, y, cv=cv, scoring="accuracy")
GridSearchCV — 超參數搜尋
from sklearn.model_selection import GridSearchCV param_grid = { 'max_depth': [3, 5, 7], 'n_estimators': [50, 100, 200] } # 3×3=9種組合，每種做5-fold CV → 跑45次 grid = GridSearchCV(RandomForestClassifier(), param_grid, cv=5) grid.fit(X_train, y_train) grid.best_params_ # {'max_depth': 5, 'n_estimators': 100} grid.best_score_ # 最佳交叉驗證分數

Keras 神經網路 Deep Learning
Activation 選哪個？
位置任務選這個數學式
隱藏層學習特徵relumax(0, x)
輸出層二分類sigmoid1 / (1 + e⁻ˣ)
輸出層多分類softmax各類機率和=1
輸出層迴歸不加（linear）直接輸出數字
Loss 選哪個？
任務標籤格式Loss
二分類0 或 1binary_crossentropy
多分類[0,0,1,0] one-hotcategorical_crossentropy
多分類2（整數）sparse_categorical_crossentropy
迴歸連續數字mse
二分類完整範例
from tensorflow.keras.models import Sequential from tensorflow.keras.layers import Dense, Dropout, Input model = Sequential([ Input(shape=(9,)), Dense(64, activation='relu'), Dropout(0.3), Dense(1, activation='sigmoid') ]) model.compile( optimizer='adam', loss='binary_crossentropy', metrics=['accuracy'] )
訓練與評估
history = model.fit( X_train, y_train, epochs=100, batch_size=32, validation_split=0.2 ) model.evaluate(X_test, y_test) model.predict(X_test) model.summary() # 查看架構與參數量 # 訓練歷程 history.history["loss"] history.history["val_loss"] history.history["accuracy"]
參數量計算公式（常考）
# 公式：(上層神經元數 × 本層神經元數) + 本層神經元數（bias） # 範例：輸入9個特徵 → Dense(10) params = (9 × 10) + 10 = 100 # Dense(10) → Dense(10) params = (10 × 10) + 10 = 110 # Dense(10) → Dense(1) params = (10 × 1) + 1 = 11
常見考試陷阱：多分類要用 softmax，不是 sigmoid；sigmoid 的數學式是 1/(1+e⁻ˣ)，不是 relu；Dropout 隨機關掉神經元是防過擬合，不是特徵選擇。

考前速記卡
fit = 學習（只在 train 做） transform = 套用（train 和 test 都做） fit_transform = 合併（只在 train 用） axis=0 = 沿列方向（往下）→ 算每欄的統計值 axis=1 = 沿欄方向（往右）→ 算每列的統計值 relu = max(0, x) 隱藏層 sigmoid = 1/(1+e⁻ˣ) 輸出層（二分類） softmax = 各類機率和=1 輸出層（多分類） Bagging = 獨立平行，投票 → 降 Variance（Random Forest） Boosting = 串行修錯，加權 → 降 Bias（XGBoost） Ridge = L2，係數縮小不歸零 Lasso = L1，係數可歸零（特徵選擇） cv=5 整數 = 自動用 StratifiedKFold（與明確建立效果相同） scoring="accuracy" ≠ scoring="f1"（注意題目要哪個）
